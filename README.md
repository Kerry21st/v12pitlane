## Статус проекта

**Проект приостановлен.**

Данный проект изначально разрабатывался как дипломная работа, предполагающая интеграцию с открытыми API для получения актуальных данных о Формуле 1 в реальном времени: расписание гонок, результаты сессий, статистика пилотов и команд, данные телеметрии.

В связи с ограничением доступа к необходимым API-сервисам на территории Российской Федерации, дальнейшая разработка проекта поставлена на паузу до момента появления альтернативных источников данных или изменения технических возможностей.

На данный момент проект представляет собой визуальную оболочку (front-end часть) с реализованной структурой и адаптивным интерфейсом. Бэкенд-часть и интеграция с API не завершены.

## Важное уведомление об использовании товарных знаков

**Данный проект носит исключительно учебный и некоммерческий характер.**

В проекте используются:
- Официальные логотипы команд Formula 1 (Red Bull Racing, McLaren, Ferrari, Mercedes и др.)

- **Команды Formula 1** — права на свои логотипы и визуальную айдентику
- **Фотографы и агентства** — права на фотографии пилотов и болидов

Проект использует указанные материалы **исключительно в демонстрационных и учебных целях**, без какой-либо коммерческой выгоды. Если вы являетесь правообладателем и считаете, что использование материалов нарушает ваши права, пожалуйста, свяжитесь с автором проекта для оперативного устранения замечаний.

## О проекте

V12Pitlane — это учебный проект, задуманный как полноценная платформа для поклонников Формулы 1 с возможностью отслеживания актуальной статистики, результатов гонок и информации о пилотах в реальном времени.

Проект включает в себя:
- Информацию о действующих пилотах и командах
- Статистику чемпионата
- Новости мира Формулы 1
- Медиагалерею
- Личный кабинет пользователя

---

## Причина приостановки разработки

- Недоступность открытых API (Ergast API, OpenF1 и др.) на территории РФ
- Отсутствие легальных альтернативных источников данных для тестирования
- Технические ограничения, не позволяющие продолжить разработку в запланированном объеме

## Установка и запуск

1. Клонируйте репозиторий:
```
git clone https://github.com/ВАШ_ЛОГИН/v12pitlane.git
```
Перейдите в папку проекта:

```
cd v12pitlane
```
Откройте файл index.html в любом современном браузере.

Примечание: Для корректного отображения фонового видео на главной странице требуется наличие файла assets/media/video/back-video.mp4. Если видео отсутствует, страница будет отображаться с черным фоном.

## Планы на будущее

Разработка проекта может быть возобновлена при появлении следующих условий:

- Доступных API-сервисов для получения данных о Формуле 1 (или возможности использования прокси-серверов)
- Альтернативных источников данных для тестирования
- Изменения технических условий для завершения дипломной работы

### В случае возобновления разработки планируется:

| № | Задача | Описание |
|---|--------|----------|
| 1 | Интеграция с API | Подключение к Ergast API, OpenF1 или альтернативным источникам для получения актуальных данных о пилотах, командах, расписании и результатах гонок |
| 2 | Реализация бэкенд-части | Разработка серверной логики на Node.js или Python для обработки запросов, кэширования данных и взаимодействия с API |
| 3 | Создание базы данных | Проектирование и реализация базы данных пользователей, избранного, истории просмотров и сохраненных настроек |
| 4 | Добавление страницы с расписанием гонок | Отображение календаря сезона, времени стартов, результатов квалификаций и гонок в реальном времени |
| 5 | Система уведомлений | Настройка push-уведомлений о начале гонок, изменениях в расписании и важных событиях |
| 6 | Медиагалерея | Создание раздела с фотографиями, видеообзорами гонок и интервью с пилотами |
| 7 | Турнирная таблица | Реализация динамической таблицы чемпионата с обновлением позиций после каждой гонки |
| 8 | Личный кабинет | Расширение функционала: сохранение любимых пилотов и команд, настройка уведомлений, история просмотров |
